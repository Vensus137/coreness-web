Платформа для создания
AI-агентов и чат-ботов

Мультитенантная платформа для создания интеллектуальных ботов с интеграцией LLM. Логика описывается в YAML-конфигурациях — декларативный подход к разработке.

Документация GitHub

Для чего Coreness

Open-source и self-hosted решение с полным контролем над данными и инфраструктурой. Инструменты для создания production-ready AI-агентов и чат-ботов.

Декларативная конфигурация

Вся логика ботов описывается в YAML-файлах. Создавайте сложные сценарии без кода.

Мультитенантность

Один экземпляр — множество независимых ботов с полной изоляцией данных через RLS.

AI и RAG

Интеграция с OpenAI, Anthropic, Google, DeepSeek. Векторный поиск через pgvector.

Scheduled сценарии

Автоматизация по расписанию через cron-выражения. Рассылки, отчёты, уведомления.

Плагинная архитектура

Легко расширяйте функционал. Storage, transitions, placeholders из коробки.

Production Ready

Надёжная база данных, фоновые задачи, обработка ошибок. Создано для реальных нагрузок.

Примеры конфигураций

В комплекте поставляется Master Bot — готовый бот для управления платформой через Telegram.

commands.yaml
start:
  description: "Start Master Bot"
  
  trigger:
    - event_type: "message"
      event_text: "/start"
    - event_type: "callback"
      callback_data: "start"

  step:
    - action: "execute_scenario"
      params:
        scenario: "default_check"
    
    - action: "send_message"
      params:
        text: |
          {_cache.translation.start}
        inline: "{_cache.translation.start_inline}"
config.yaml
i18n:
  available_languages:
    - en
    - ru
  default_language: en

environments:
  available:
    - test
    - prod
  default: prod

deployment:
  available_modes:
    - docker
    - native
  default: docker
